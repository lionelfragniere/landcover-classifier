# UNOPS COMPASS Design System

This document outlines the design guidelines for UNOPS COMPASS applications.

## Typography
- **Body Text:** Inter (Google Fonts)
- **Headings:** Inter Tight (Google Fonts)

## Colors
- **Primary:** `#0092D1` (UNOPS Blue) - Used for CTAs, links, primary buttons
- **Supporting:**
    - `#004976` (Midnight) - Dark headers/sidebar
    - `#00A997` (Teal) - Secondary actions
- **Semantic:**
    - `#4C9F38` (Success)
    - `#FFC215` (Warning)
    - `#DA291C` (Error)
    - `#0092D1` (Info)
- **Backgrounds:**
    - `#F6F9FC` (Cool Background) - Light theme default
    - `#FFFFFF` (Cards) - Light theme cards
    - `#0D1E2F` (Dark Mode Base) - Dark theme default

## UI Elements
- **Buttons:** 8px border-radius, primary background `#0092D1` with white text
- **Cards:** 12px border-radius, white background, shadow `0 4px 12px rgba(0,73,118,0.08)`

## Logo
- Include the UNOPS logo (`public/unops-logo.png`) in headers/nav where branding applies.

## Theme
- Light theme by default, with dark mode toggle support preferred.
